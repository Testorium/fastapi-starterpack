# FastAPI Starter Pack

A minimal yet powerful FastAPI starter pack, fully Dockerized and ready for modern backend development.

## Features

- 🐍 **FastAPI** 
- 🐳 **Dockerized**
- 🐘 **PostgreSQL**
- 🖥️ **pgAdmin**

---

To start the project just execute this command:

```bash
docker compose up --build -d
```
